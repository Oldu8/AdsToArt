AdsToArt

Hey, folks!
AdsToArt is the simple extension that swaps ads with images from sets.

All you need is enable extension and select the image set.

Enjoy your time in internet!
P.S. Extension in beta. You can face some bugs and glitches. Sorry for that, but if you want to improve it, please leave a feedback.
